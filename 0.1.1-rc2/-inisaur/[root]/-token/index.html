Token

data class Token(val type: TokenType, val value: String?, val line: Int, val column: Int)(source)

Represents a token in the INI file parsing process.

Each token has a type, an optional value, and position information (line and column) within the source file. Tokens are produced by the Lexer and consumed by the Parser.
